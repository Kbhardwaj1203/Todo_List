# How to setup a React app with Vite?

-> npm create vite@latest.
-> Name the app.
-> Select react or react-ts.
-> Select JavaScript + SWC or TypeScript + SWC option.
-> cd your-project-name-here.
-> npm i.
-> npm run dev.
